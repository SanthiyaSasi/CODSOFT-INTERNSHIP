# CODSOFT-INTERNSHIP
This repository contains the projects completed during my CodSoft C++ Programming Internship
## Tasks Completed
## Task 1: Number Guessing Game

A console-based C++ game where the computer generates a random number and the user attempts to guess it. The program provides feedback after each guess, indicating whether the entered number is too high, too low, or correct.

### Features
- Random number generation
- User-friendly gameplay
- Hint system for incorrect guesses
- Unlimited attempts until the correct guess

### Technologies Used
- C++
- Standard Library

## Task 2: Simple Calculator

A C++ calculator application that performs basic arithmetic operations based on user input. The program allows users to select an operation and instantly view the calculated result.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus Operation
- Continuous calculations using loops

### Technologies Used
- C++
- Standard Library
